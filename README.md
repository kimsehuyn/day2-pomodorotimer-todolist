# Focus Flow — Pomodoro Timer & Todo List

HTML, CSS, JavaScript만으로 만든 포모도로 타이머와 할 일 목록 결합 앱입니다.

## 주요 기능

### 포모도로 타이머
- **집중 25분 · 짧은 휴식 5분 · 긴 휴식 15분** (4세션 후 긴 휴식)
- 아날로그 타이머 스타일 UI (0~55분 눈금, 파스텔 웨지 진행 표시)
- 중앙 노브 버튼으로 시작 / 일시정지
- 세션 카운터 및 초기화
- 타이머 종료 시 알림음
- 할 일과 연동 — 선택한 작업에 포모도로 횟수 자동 기록

### 할 일 목록
- 할 일 추가, 완료, 삭제
- **전체 / 진행중 / 완료** 필터
- **별표(★)** 로 중요 할 일 표시
- **카테고리** (업무 · 공부 · 생활 · 개인) — 클릭 시 순환 변경
- 완료율 원형 진행률 표시
- `localStorage`에 자동 저장 (새로고침 후에도 유지)

### 설정
- **한/영 모드** — 헤더 `KO`/`EN` 버튼으로 UI·날짜·카테고리 라벨 전환 (`focusflow-lang`)
- **다크모드** — 헤더 테마 버튼으로 전환, 최초 방문 시 시스템 설정 반영 (`focusflow-theme`)

## 개발 이력

### v1.0 — 초기 버전
- 포모도로 타이머 + 할 일 목록 기본 기능
- 원형 프로그레스 링, 다크 테마 UI
- 할 일 선택 시 현재 작업 표시, 포모도로 카운트 연동

### v2.0 — UI 개선
- **타이머**: 아날로그 다이얼 UI로 전면 교체
  - SVG 기반 0~55분 눈금 및 숫자
  - 남은 시간을 보여주는 웨지(파이 조각) — 시간 경과에 따라 축소
  - 중앙 노브 형태의 시작/일시정지 버튼
  - 모드별 웨지 색상 (집중 / 짧은 휴식 / 긴 휴식)
- **할 일**: Todolize 스타일 라이트 테마 적용
  - 오늘의 할 일 헤더 + 완료율 미니 차트
  - 필터 탭, 별표, 카테고리 컬러 바
  - 둥근 체크박스 및 카드형 리스트

### v2.1 — 파스텔 톤 · 한/영 · 다크모드 (현재)

#### 주요 내용
- **파스텔 색감**: 전체 UI를 더스티 로즈 / 세이지 / 파우더 블루 계열로 통일
- **한/영 모드**: `I18N` 사전 + `data-i18n`으로 정적·동적 문자열 일괄 전환
- **다크모드**: `html[data-theme="dark"]` CSS 변수 오버라이드 (파스텔 톤 유지)
- 헤더에 언어·테마 토글 버튼 추가
- 언어·테마 설정을 `localStorage`에 저장해 새로고침 후에도 유지

#### 오류 수정 사항

| 항목 | 내용 |
|------|------|
| 빈 목록 문구 하드코딩 | CSS `content`에 한국어가 고정되어 영어 모드에서 안 바뀌던 문제 → `data-empty` + `attr()`로 언어별 표시 |
| 다이얼 색상 하드코딩 | 다크모드에서 다이얼·노브가 밝게 남던 문제 → CSS 변수(`--dial-*`, `--knob-*`)로 분리해 테마별 적용 |
| 카테고리 라벨 미갱신 | 언어 전환 시 할 일 카테고리 텍스트가 그대로이던 문제 → `applyLanguage()`에서 `renderTodos()` 재호출 |
| 날짜 로케일 고정 | 날짜가 항상 `ko-KR`로 표시되던 문제 → 현재 언어에 맞춰 `en-US` / `ko-KR` 전환 |
| 테마 최초값 | 저장된 테마가 없을 때 시스템 `prefers-color-scheme`를 반영하도록 처리 |

### 이전 오류 수정 (v2.0)

| 항목 | 내용 |
|------|------|
| 다이얼 눈금 중복 | 1분 단위 눈금(tick)과 점(dot)이 겹쳐 보이던 문제 → tick만 사용하도록 정리 |
| 기존 데이터 호환 | `localStorage`에 `starred`, `category`가 없을 때 기본값(`false`, `work`) 자동 적용 |
| 완료된 할 일 선택 | 완료된 할 일이 현재 작업으로 남아 있던 경우 → 선택 해제 및 UI 동기화 |
| 모드 전환 중 타이머 | 타이머 실행 중 모드 탭 클릭 시 상태 꼬임 방지 → 실행 중에는 모드 변경 불가 |
